◆ DacSetGain()

trap_voltsense_result DacSetGain ( dac_buffer_gain  gain)

Sets the DAC buffer gain to the value supplied. Note the DAC buffer gain will be either 1 or 0.9.

PIO setup is not handled by this trap but can be done at any point. XIO 12 needs to be configured as analogue pin. This is done by using PioSetMapPins32Bank to put the PIO under HW control and then PioSetFunction to set the function to XIO_ANALOGUE.

Parameters
gaingain, 0.9, 1
Returns
result, success, or failure with reason

◆ XioMeasureVoltage()

trap_voltsense_result XioMeasureVoltage ( Task  task,
xio_source  first_xio,
xio_source  second_xio,
vga_gain  voltage_gain 
)

API to measure voltage on an XIO. PIO setup is not handled by this trap but can be done at any point. XIO pads need to be configured as analogue pins. This is done by using PioSetMapPins32Bank to put the PIO under HW control and then PioSetFunction to set the function to XIO_ANALOGUE.

Parameters
taskThe task to which the message containing the voltage reading will be delivered
first_xioxio or first xio in difference pair to use for input
second_xioxio or second xio in difference pair to use for input
voltage_gainVGA value, 0.85, 1, 2, 4, 8
Returns
result, success, failure, failure with reason
