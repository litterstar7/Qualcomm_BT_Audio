◆ SdMmcSlotInit()

bool SdMmcSlotInit ( bool  init)

Initialise/deinitialise SD host slot driver. Initialisation will be done of the slots for which MIB key is set (SDHostSlot0 and SDHostSlot1). If none of the key is set, no operation will be performed but trap will return TRUE. It will return FALSE if initialisation operation fails. For deinitialisation; all initialised slots will be deinitialised. If no slot was initialised, no operation will be performed but trap will return TRUE. It will return FALSE if deinitialisation operation fails. On QCC512x, QCC302x and QCC303x, once SD host is initialised, VmRequestRunTimeProfile() changes will pend until the SD host slot driver is deinitialised by this API.

Parameters
initInitialise (TRUE) or Deinitialise (FALSE) slot driver.
Returns
True if operation is successful, otherwise FALSE.
