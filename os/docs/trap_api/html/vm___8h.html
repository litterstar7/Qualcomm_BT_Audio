typedef unsigned int conn_context_t

Opaque connection context type. Used in VmOverrideSyncConnContext, VmOverrideRfcommConnContext, VmOverrideL2capConnContext traps. This type is the same as Bluestack's context_t. It is redefined here to avoid type clashes between bluestack types and VM types.
