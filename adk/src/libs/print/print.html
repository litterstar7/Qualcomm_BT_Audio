Print Library

This library allows the user to add diagnostic messages to their code and enable/disable them with a single switch in the makefile.

PRINT(("Hello world arg %d", arg));

This will generate a call to printf if and only if DEBUG_PRINT_ENABLED macro was defined when the <print.h> header file was included.

The CPRINT macro is also provided; it expands to call the cprintf function described in debug output.
