Detailed Description

Header file for the GATT Transport Discovery Service library.

This file provides documentation for the GATT Transport Discovery Service library API (library name: gatt_transport_discovery_server).

This API is used to return a TDS client configuration value to the library when a GATT_TRANSPORT_DISCOVERY_SERVER_READ_CLIENT_CONFIG_IND message is received.

Parameters
tdsThe pointer that was in the payload of the GATT_TRANSPORT_DISCOVERY_SERVER_READ_CLIENT_CONFIG_IND message.
cidThe connection identifier from the GATT_TRANSPORT_DISCOVERY_SERVER_READ_CLIENT_CONFIG_IND message.
client_configThe client configuration to return to the library
Returns
TRUE if success, FALSE otherwise

This API is used to send an indication to a remote Transport Discovery client regarding the write to the Control Point. This will
only be allowed if notifications have been enabled by the remote device.

Parameters
tdsThe instance pointer that was passed into the GattTransportDiscoveryServerInit API.
cidThe connection identifier from the GATT_TRANSPORT_DISCOVERY_SERVER_WRITE_CONTROL_POINT_IND message.
tds_ind_sizeThe length of the Transport Discovery indication
tds_ind_dataThe Transport Discovery indication itself
Returns
TRUE if success, FALSE otherwise
