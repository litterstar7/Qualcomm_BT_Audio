Detailed Description

Header file for the GATT ama server library.

This file provides documentation for the GATT ama server library API (library name: gatt_ama_server).

Typedef Documentation

◆ GAMASS

typedef struct __GAMASS GAMASS

The ama server internal structure for the server role.

This structure is visible to the application as it is responsible for managing resource to pass to the ama library. The elements of this structure are only modified by the ama library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.
