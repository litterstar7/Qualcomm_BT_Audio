void HidConnect ( HID_LIB *  hid_lib,
Task  task,
const bdaddr *  bd_addr,
uint16  hid_conftab_length,
const uint16 *  hid_conftab 
)

Called to initiate a new connection,.

Parameters
hid_libHID library structure.
TaskTask that connection should be associated with.
bd_addrBluetooth address of device to connect to.
hid_conftab_lengthThe length of the L2CAP configuration table data array.
hid_conftabPointer to a L2CAP configuration table of uint16 values. These are key value pairs defining configuration options to be passed to Bluestack. This pointer will be passed to Bluestack at which time the VM memory slot for it will be freed.

Setting both hid_conftab_length and hid_conftab to 0 will cause the default L2CAP configuration to be used.
