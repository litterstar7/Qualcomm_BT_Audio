◆ TransportMgrCreateTransLinkData()

bool TransportMgrCreateTransLinkData ( Task  application_task,
transport_mgr_link_cfg_t  link_cfg,
Sink  snk 
)

Create RFCOMM link data on new Primary during a handover commit. If rfcomm channel provided in link_cfg parameter is already present in a node, it is updated with the provided sink and function returns TRUE, else new node is created and function returns FALSE.

If TRUE is returned, new RFCOMM channel should be registered (since current is used up) and current SDP record should be removed and new created.

If FALSE is returned, current SDP record should not be changed.

Parameters
application_taskGAIA's application task
link_cfg: link conifguration (includes RFCOMM server channel)
snkSink
@return bool
