◆ TaskList_IterateWithDataRaw()

bool TaskList_IterateWithDataRaw ( task_list_t *  list,
Task *  next_task,
task_list_data_t **  raw_data 
)

Iterate through all tasks in a list returning raw data stored in the task_list_t.

Useful for efficiently modifying the data for all items in a task_list_t.

Pass NULL to next_task to start iterating at first task in the list. On each subsequent call next_task should be the task previously returned and TaskList_Iterate will return the next task in the list.

Parameters
[in]listPointer to a Tasklist.
[in,out]next_taskPointer to task from which to iterate.
[out]raw_dataAddress of pointer to the task's raw data stored by tasklist.
Returns
bool TRUE next_task is returning a Task. FALSE end of list, next_task is not returning a Task.
Note
The lifetime of the raw_data pointer is limited. The task_list_t module reserves the right to invalidate this pointer (e.g. through reallocation) in any subsequent task_list_t function call.
