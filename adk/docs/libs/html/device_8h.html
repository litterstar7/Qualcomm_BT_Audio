Detailed Description

A device instance that represents a collection of profiles/services/etc.

Copyright
Copyright (c) 2018-2019 Qualcomm Technologies International, Ltd.
All Rights Reserved.
Qualcomm Technologies International, Ltd. Confidential and Proprietary.

A device will usually be a connected remote device connected to the local device, but it could also be used to store the properties of the local device.

◆ Device_SetProperty()

bool Device_SetProperty ( device_t  device,
device_property_t  id,
const void *  value,
size_t  size 
)

Set a property for a device by copying the data it contains.

When the property is set, it is copied from #value to a buffer owned by the property.

This function will fail if the property already exists or the device cannot hold another property.

If size is too large or there is not enough memory to allocate a buffer to copy the data into this function will panic.

Parameters
deviceDevice to set the property for.
idProperty to set.
valuePointer to the location to copy the data from.
sizeSize of the data in bytes pointed to by #value
Returns
TRUE if the property was set ok; FALSE otherwise.
