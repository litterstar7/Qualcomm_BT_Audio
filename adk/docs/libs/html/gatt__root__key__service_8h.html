Detailed Description

Header file for the GATT root key service.

This file provides documentation for types that are used for the service and so common to the client and the server.

enum gatt_root_key_challenge_status_t

Status code returned following a GattRootKeyChallengePeer().

The status is return in a GATT_ROOT_KEY_CLIENT_CHALLENGE_IND_T message.

Enumerator
gatt_root_key_challenge_status_success 

Indicates that the challenge implemented in the GATT Root Key Transfer Service has completed successfully

gatt_root_key_challenge_status_fatal 

Indicates that the challenge implemented in the GATT Root Key Transfer Service has completed, but has failed. This can be used as an indication that the peer device can be considered untrustworthy
