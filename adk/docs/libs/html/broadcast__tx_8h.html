◆ broadcast_tx_state

enum broadcast_tx_state

Definition of broadcaster states.

NOTE - transition states MUST have odd values, as the state machine event handler uses this to identify transition states and set/clear a message lock to control message delivery into the state machine. See broadcastTxSetState().

Enumerator
broadcastTxStateInit 

The broadcaster is initialised.

broadcastTxStateConfiguring 

The broadcaster is (re)configuring. Transition State.

broadcastTxStateIdle 

The broadcaster has no CSB or Sync Train active.

broadcastTxStateStartingCsb 

The broadcaster is starting a broadcast. Transition State.

broadcastTxStateCsb 

The broadcaster has an active broadcast.

broadcastTxStateStoppingCsb 

The broadcaster is stopping a broadcast. Transition State.

broadcastTxStateCsbSyncTrain 

The broadcaster has an active broadcast and sync train.

broadcastTxStateCsbStartingSyncTrain 

The broadcaster has an active broadcast and is starting the sync train. Transition State.

broadcastTxStateDestroyingInstance 

An instance of the library in the broadcast role is being destroyed. Transition State.
