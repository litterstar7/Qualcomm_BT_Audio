typedef enum root_key_server_state_t root_key_server_state_t

Enumerated type for the internal state of the root key server.

The state controls what operations are allowed & what messages are expected.

Unexpected messages cause a jump to the error state, as this service is secure.

Unexpected commands may just be ignored.
