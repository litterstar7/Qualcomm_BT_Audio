Detailed Description

This file provides the functions required to send NFC primitives to NFC subsystem. .i.e. the API between the VM application and the NFC stack on P0.

◆ NfcSendCtrlRegisterReqPrim()

void NfcSendCtrlRegisterReqPrim ( void  )

#include <nfc_api/nfc_api.h>

Sends a NFC_CTRL_REGISTER_REQ primitive to the NFC subsystem.

After this message has been sent the NFC task handler will be able to receive NFC messages according to its configuration. For instance:

  • config
    1. NONE only a single message (e.g. NFC_CTRL_READY_IND) can be received.
    2. TT2 NFC_CTRL_CARRIER_ON_IND, NFC_CTRL_CARRIER_LOSS_IND ...

◆ NfcSetRecvTask()

void NfcSetRecvTask ( TaskData *  nfcTask)

#include <nfc_api/nfc_api.h>

Sets the NFC Recv Task.

Parameters
nfcTaskData passed to the NFC Recv task. 
The nfcTask task will be called for every NFC message received from the 
NFC subsystem located on P0.

nfcTask is usually set by default to NFC_CL task as the NFC API messages 
are handled by default in NFC_CL task.

@warning if NFC_CL task is overridden by the main task then
#define NFC_MESSAGE_BASE 0x7E00 SHOULD NOT be use by other base.
