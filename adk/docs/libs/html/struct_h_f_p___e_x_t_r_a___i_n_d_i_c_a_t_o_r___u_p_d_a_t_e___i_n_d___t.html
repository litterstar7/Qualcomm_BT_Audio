Detailed Description

This message returns the value of an indicator the AG supports.

If the application registers that it is interested in a particular indicator (other than the ones defined by the HFP specification) it will be sent this message whenever an indicator status update (+CIEV) is received from the AG. All indicator updates will be sent to the application even if they are not for the indicator the application initially registered an interest in. It is the application's responsibility to determine whather the indicator update is of interest.
