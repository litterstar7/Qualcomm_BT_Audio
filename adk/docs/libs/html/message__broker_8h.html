Detailed Description

Interface to the Message Broker.

All Rights Reserved.
Qualcomm Technologies International, Ltd. Confidential and Proprietary.

The Message Broker allows client modules to register interest in System Messages by Message Group. The Message Broker registers with the message group owner on behalf of the client module. The message group owner subsequently messages the client module directly. The message group owner's messge group ID and registration function is provides to message broker in MessageBroker_init().
