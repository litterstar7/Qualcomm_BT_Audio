◆ GATT_SERVER_SERVICE_CHANGED_INDICATION_CFM_T

typedef struct PACK_STRUCT __GATT_SERVER_SERVICE_CHANGED_INDICATION_CFM GATT_SERVER_SERVICE_CHANGED_INDICATION_CFM_T

Contains status of the send Service Changed Indication request.

This message is received in response to GattServerSendServiceChangedIndication(). When an Indication is successfully sent, this message is returned with status 'gatt_status_success_sent'. Then, when the remote device responds to the notification, this message should be returned with status 'gatt_status_success'.

Parameters
statusindicates the outcome of the send indication request.
cidConnection identifier of remote device.

◆ GGATTS

typedef struct __GGATTS GGATTS

The GATT Server internal structure for the server role.

This structure is visible to the application as it is responsible for managing resource to pass to the "GATT Server" library. The elements of this structure are only modified by the "GATT Server" library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.
