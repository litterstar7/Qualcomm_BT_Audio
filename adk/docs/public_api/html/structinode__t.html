Detailed Description

Record the position and length of a file extent in the file system.

A file consists of one or more extents 16-bit extents good for 4K sectors up to max 256MB (inode size=4) 8-bit extents good for 4K sectors up to max 1MB (inode size=2) An inode = { 0xff,0xff } is unused (so far) An inode can have a length of 0xff, but it can't also start at 0xff. An inode = { 0x00,0x00 } is to be ignored (it also means it's deleted) An inode can start at 0x00, but it can't also have a length of 0x00. The minimum useful length is 0x01.
