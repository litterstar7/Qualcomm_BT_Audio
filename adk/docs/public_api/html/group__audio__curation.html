Detailed Description

A component responsible for controlling audio curation services.

Responsibilities:

  • Handles ANC UI inputs: ANC Power On, Off, ANC On, Off, Toggle etc which in-turn calls audio anc domain interfaces.

The Audio Curation uses audio_anc_domain Audio Anc domain and UI UI domain.
