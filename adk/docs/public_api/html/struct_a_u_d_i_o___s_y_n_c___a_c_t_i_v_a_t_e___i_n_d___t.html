Detailed Description

Request a change to "Active" state.

Sent by the audio source to request that the audio sink does what is needed to get to the AUDIO_SYNC_STATE_ACTIVE state.

The audio sink must send a AUDIO_SYNC_ACTIVE_RES in response, even if it cannot reach the requested state.
