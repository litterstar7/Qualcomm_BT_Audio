Detailed Description

Provides a very simple read/write file system interface to store, retrieve and remove user content.

rafs_errors_t Rafs_Compact ( Task  task,
const char *  partition_name 
)

#include <adk/src/services/voice_ui/rafs/rafs.h>

Trigger compaction of the FAT table and/or the files.

Parameters
[in]taskThe task that will receive the MESSAGE_RAFS_COMPACT_COMPLETE message. This may be NULL, but the application would have to deal with
See also
RAFS_BUSY errors in the meantime. RAFS message IDs are not globally unique, a dedicated handler is required.
Parameters
[in]partition_nameIdentify the partition to be compacted.
Returns
RAFS_OK or an error code.
Note
If the result is an error, no MESSAGE_RAFS_COMPACT_COMPLETE is sent.
This only compacts the FAT table in this implementation.
This will modify the sequence count of files.

Open a directory to read information about each file.

Parameters

[out]dir_idA pointer to where to store the opened directory id.
[out]numA pointer to where to store the number of directory entries in the opened directory.
