◆ source_sync_input_buffer_size_samples

uint16 source_sync_input_buffer_size_samples

The size of the buffer at the input of the source sync in samples.

If the source sync version is >3.3, the source sync capability supports setting of its input terminal buffer size, so a separate basic passthrough operator is not required before the source sync to set a buffer size.

For source sync version <=3.3, if a buffer is required before the source sync, a basic passthrough operator may be added.

If this parameter is non-zero, when configuring the output chain, if a basic passthrough operator is found, its buffer size will be set to the value of this parameter and the source sync input terminal buffer size will not be set. If no basic passthrough operator is found the buffer size will be in the source sync terminal buffer size. This will fail if the source sync version is <=3.3.
