◆ MicDisconnectIndication

bool(* MicDisconnectIndication) (const mic_disconnect_info_t *info)

Mics will be disconnected in case of a concurrency. The reason is delivered in *info. The client returns TRUE if it wants to reconnect and FALSE if it wants to stop.
