Detailed Description

A component that dispatches the connection library messages to interested clients.

The connection library sends many messages to single task (the task passed into ConnectionInit). The purpose of Connection Message Dispatcher is to redirect those messages to interested clients.

Instead of resending messages the client's task handler is called directly.

To limit number of client, clients register a task for a message group, not an individual message.

The mapping of the messages to groups in hard-coded in this component.
