void EarbudInit_CompleteInitialisation ( void  )

This function is used to complete the initialisation process.

This should be called on receipt of the #APPS_COMMON_INIT_CFM message.

This function is called to let the application initialisation module know that the common initialisation is completed. It finalises and completes the initialisation.
